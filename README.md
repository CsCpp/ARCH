# ARCH (ARC-CORE H-Bridge) TIG AC/DC Controller

Профессиональный контроллер для сварочного аппарата TIG AC/DC на базе микроконтроллера STM32F103. Разработан для управления мощным H-мостом в условиях сильных электромагнитных помех (ВЧ-поджиг).

## ⚡ Аппаратная часть (Hardware)
- **Силовые ключи:** 4x IGBT-модуля Mitsubishi CM600HA-24H (600А / 1200В).
- **Питание силовой шины:** 80 В DC от базового инвертора.
- **Драйверы затворов:** Промышленные платы на базе ACPL-316J с транзисторным Totem-Pole буфером.
  - Питание драйверов: Изолированное двуполярное +15В / -8В (3 раздельных канала).
  - Защита: Аппаратная защита от десатурации (DESAT) с выводом сигнала FAULT на пин `TIM1_BKIN` (экстренное глушение ШИМ).
  - Затворные резисторы: ~4.7 Ом (рекомендуется разделение диодом на Turn-on / Turn-off).
- **Датчик тока:** ABB ES1000-9676 (Closed Loop, 1:1000).
  - Схема включения: Измерение истинного AC-тока (на выходе моста).
  - Нагрузочный резистор ($R_m$): 6.2 Ом (2 Вт).
  - Смещение для АЦП: Прецизионный делитель на 1.65 В.
- **Интерфейс (UI):** 4-разрядный 7-сегментный индикатор (динамическая индикация) + Энкодер с кнопкой. Выбран для максимальной надежности при работе в крагах и защиты от помех.

## 🧠 Программная часть (Software)
- **Управление H-мостом:** Таймер `TIM1` (16 кГц) с генерацией аппаратного Dead-Time.
- **Генерация AC:** Таймер `TIM2`. Прерывание переключает диагонали моста, формируя заданную частоту (20-250 Гц) и баланс полярности (10-90%).
- **Регулировка тока:** ПИ-регулятор с частотой обновления 1 кГц.
- **Защита от осциллятора:** Алгоритм "Безопасный старт" отключает ШИМ на момент пробоя ВЧ-искры.
- **Память:** Безизносное сохранение структуры настроек в последнюю страницу Flash-памяти МК.

## 🔄 Сварочный цикл (State Machine)
1. **IDLE:** Ожидание нажатия кнопки горелки.
2. **PRE-GAS:** Продувка газом (ШИМ строго выключен).
3. **IGNITION:** Удар осциллятора (до появления тока > 5А).
4. **ARC START:** Горячий старт (повышенный ток для пробития оксидной пленки).
5. **WELDING:** Основной рабочий режим с ПИ-регулированием.
6. **DOWN SLOPE:** Плавное заваривание кратера.
7. **POST-GAS:** Остывание вольфрамового электрода.
