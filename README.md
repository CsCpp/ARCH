# ⚡ ARCH - TIG Welding Controller

**Версия:** 4.4  
**Платформа:** STM32F103C8T6 (Blue Pill)  
**Тип устройства:** Контроллер инверторного аппарата аргонодуговой сварки (TIG AC/DC).

## 📖 Описание
Проект **ARCH** — это микропроцессорная система управления сварочным инвертором. Она обеспечивает генерацию ШИМ-сигналов для силового моста, контроль сварочного тока по ПИД-закону, управление газовой автоматикой, осциллятором поджига дуги и пользовательским интерфейсом (энкодер + 7-сегментный дисплей).

## ✨ Ключевые особенности
* **Hardware PWM:** Аппаратное управление стойками инвертора через `TIM1` с комплементарными выходами (High/Low side) и аппаратной вставкой мертвого времени (Dead-Time).
* **Ultra-Fast Protection:** Мгновенная блокировка генерации ШИМ и отключение силового буфера 74HCT541 при срабатывании защиты драйверов (через аппаратное прерывание `EXTI`).
* **Linear Display Bus:** Сверхбыстрая динамическая индикация на 4-разрядном дисплее (Общий Анод). Данные выводятся в порт B за одну тактовую операцию сдвига регистра.
* **Smart UI:** Настройка параметров (ток, частота, баланс) с помощью энкодера. Автоматическое сохранение настроек во встроенную Flash-память микроконтроллера.
* **PID Control:** Обратная связь по току с использованием датчика Холла (например, ABB ES1000).

## ⚙️ Шпаргалка по настройке STM32CubeMX
Для корректной работы прошивки периферия должна быть настроена следующим образом:

1. **System Core -> SYS:** Debug: `Serial Wire` (остальные выводы JTAG отключаются в коде для работы PB3/PB4).
2. **Timers -> TIM1 (PWM):** - Channel 1: `PWM Generation CH1 CH1N`
   - Channel 2: `PWM Generation CH2 CH2N`
   - *Включить Dead Time по необходимости для ваших IGBT.*
3. **Timers -> TIM2 (AC AC/DC Phase):** Base timer, включить прерывание (Priority 1).
4. **Timers -> TIM3 (Display):** Base timer, включить прерывание (Priority 2).
5. **Analog -> ADC1:** IN4 -> `Single-ended`. Sampling Time установить на `28.5` или `55.5` Cycles для фильтрации шума. Прерывания выключить.
6. **System Core -> GPIO (EXTI):** - PB1 настроить как `GPIO_EXTI1`.
   - Режим: `Rising edge trigger detection`.
   - Включить в NVIC: `EXTI line1 interrupt` с наивысшим приоритетом (Priority 0).

## ⚠️ Важные замечания по безопасности
* **ЭМС (Электромагнитная совместимость):** Сварочный инвертор генерирует сильные помехи. Обязательно используйте RC-фильтры на входах энкодера, кнопки горелки (PA0-PA3) и линии защиты (PB1). Рекомендуется керамика `0.1uF` максимально близко к пинам МК.
* **Гальваническая развязка:** Линии управления анодами дисплея (PB12-PB15) и силовые сигналы ШИМ **должны** быть оптически или гальванически развязаны от силовой земли во избежание пробоя контроллера.
* **Initial State:** При подаче питания аппаратный пин `PB11` программно подтягивается к высокому уровню (`HIGH`), чтобы заблокировать силовой буфер до тех пор, пока система и таймеры не будут полностью инициализированы.
