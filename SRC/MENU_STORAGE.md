# 📖 Техническое описание: Меню и Flash-память (TIG Control v1.0)

Этот документ содержит описание архитектуры пользовательского интерфейса и механизма работы с энергонезависимой памятью в контроллере сварочного инвертора на базе STM32F103.

---

## 💾 1. Работа с Flash-памятью (Эмуляция EEPROM)

В данной реализации настройки сохраняются напрямую во внутреннюю Flash-память микроконтроллера, так как STM32F103 не имеет встроенной EEPROM.

### Технические параметры хранения:
* **Адрес страницы:** `0x0800FC00` (Последняя 63-я страница для версии C8T6 64KB).
* **Метод записи:** Half-word (16-бит). 
* **Алгоритм:** Перед записью страница стирается целиком (1 КБ).
* **Контроль целостности:** Используется `SETTINGS_MAGIC` (`0xABCD`). Если при чтении ключ не совпадает (чистая память), загружаются безопасные значения по умолчанию.

### Карта распределения памяти (Memory Map):
| Смещение | Тип данных | Описание |
| :--- | :--- | :--- |
| `0x00` | `uint16_t` | Magic Key (`0xABCD`) |
| `0x02` | `uint16_t` | Установленный рабочий ток (`set_amp`) |
| `0x04` | `uint16_t` | Индекс активного режима (`cur_mode`) |

## 🕹️ 2. Логика интерфейса (GUI FSM)

Управление реализовано в виде конечного автомата (Finite State Machine) с тремя основными состояниями:

### A. Режим WORK (Основной экран)
* **Отображение:** Текущий ток (например, `A120`).
* **Энкодер:** Регулировка силы тока с шагом 5А.
* **Кнопка (коротко):** Переключение пресетов: `FE` (Сталь), `SS` (Нержавейка), `CO` (Cold Weld), `SP` (Spot), `AL` (Алюминий).
* **Кнопка (длинно):** Переход в режим настроек профиля (`MENU_SEL`).

### B. Режим MENU_SEL (Навигация по параметрам)
* **Отображение:** Текстовая аббревиатура параметра (например, `Pr G`).
* **Энкодер:** Выбор нужного параметра из списка.
* **Кнопка (коротко):** Вход в редактирование выбранного значения.
* **Системные пункты:**
    * `5AvE` — Фиксация изменений во Flash и выход.
    * `E5c` — Отмена изменений и возврат в рабочий режим.

### C. Режим MENU_SET (Редактирование значения)
* **Отображение:** Мигающее числовое значение параметра.
* **Энкодер:** Инкремент/декремент значения.
* **Кнопка (коротко):** Сохранение во временную переменную и возврат в `MENU_SEL`.

## 📊 3. Список настраиваемых параметров

| Код | Параметр | Диапазон | Описание |
| :--- | :--- | :--- | :--- |
| `Pr G` | Pre-Gas | 0.1 - 2.0 с | Продувка газа перед дугой |
| `St A` | Start Amp | 10 - 100 А | Ток поджига |
| `UP S` | Up Slope | 0.0 - 5.0 с | Время нарастания тока |
| `dn S` | Down Slope | 0.0 - 10.0 с | Время спада (заварка кратера) |
| `Po G` | Post-Gas | 1.0 - 20.0 с | Защита шва после сварки |
| `t Ur` | Pulse Time | 10 - 500 мс | Время горения (для пульса) |
| `t PA` | Pulse Pause | 10 - 999 мс | Время паузы (для пульса) |
| `FrEq` | AC Frequency| 20 - 200 Гц | Частота переменного тока |
| `bALn` | AC Balance | 10 - 50 % | Баланс (очистка алюминия) |

## 🔠 4. Шрифты и символы (7-Seg)

Для вывода букв на сегментный индикатор используется специфический маппинг в `display_driver.c`:
* `G` -> Отображается как `C` с чертой (индекс 15).
* `S` -> Отображается как цифра `5` (индекс 14).
* `r` -> Малая буква (индекс 17).
* `n` -> Похожа на перевернутую `u` или малую `n`.
