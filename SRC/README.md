# TIG Welding Controller (STM32F103)

Система управления сварочным инвертором с поддержкой режимов AC/DC и импульсной сварки.

## Основные возможности
- **Режимы:** MMA (FE), Нержавейка (Pulse), Холодная сварка (COLD), Точечная (SPOT), Алюминий (AC).
- **Настройка циклограммы:** Предгаз, нарастание тока, пиковый ток, спад тока, постгаз.
- **AC Balance:** Регулировка частоты (20-200Гц) и баланса полярности.
- **Сохранение:** Автоматическое сохранение параметров во внутреннюю Flash-память.

## Железо (Pinout)
- **PA8/PA9:** ШИМ верхних плечей моста (TIM1_CH1, TIM1_CH2).
- **PA7/PB0:** Управление нижними плечами (1N/2N драйверы).
- **PA5:** Реле газа.
- **PA6:** Осциллятор (HF Start).
- **PA0:** Кнопка горелки (Активный 0).
- **PA4:** Датчик тока (АЦП).
- **PB12-PB15:** Катоды дисплея (через транзисторы).
- **PA10-PA15/PB3-PB4:** Сегменты дисплея (через резисторы 220 Ом).

## Управление
1. **Главный экран:** Поворот энкодера — изменение силы тока (А).
2. **Короткое нажатие:** Переключение между 5 предустановленными режимами.
3. **Длинное нажатие (1.2 сек):** Вход в меню детальных настроек.
   - Выбор параметра поворотом.
   - Короткое нажатие для редактирования (значение начнет мигать).
   - `5AvE` — сохранить и выйти.
   - `E5c` — выйти без сохранения.

## Сборка
Проект настроен для STM32CubeIDE или Keil uVision. 
Частота ядра: 72 МГц. 
Частота ШИМ: 16 кГц.
