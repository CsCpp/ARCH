## 📍 Основные возможности
* **5 Режимов сварки:** Steel (FE), Stainless (SS), Cold Weld (CO), Spot (SP), Aluminum (AL).
* **Энергонезависимая память:** Автоматическое сохранение последнего выбранного тока и режима во Flash.
* **Управление полным мостом:** Реализовано на TIM1 (Partial Remap) с аппаратной защитой (Break).
* **Интуитивное меню:** 4-разрядный 7-сегментный дисплей и энкодер.

## 🔧 Аппаратная часть (Pinout)
- **Дисплей:** PB12-15 (Аноды), PB3-9 (Сегменты). *Важно: JTAG отключен программно для использования PB3/PB4.*
- **Сила (PWM):** PA8, PA9 (Верх), PA7, PB0 (Низ).
- **Периферия:** PA0 (Кнопка горелки), PA5 (Газ), PA6 (Осциллятор).
